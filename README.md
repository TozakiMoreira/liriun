# Jarvis

Organizador pessoal de ideias e tarefas com IA.

## Estrutura do repositório

```
Jarvis/
├── backend/      # API .NET 8 (ASP.NET Core, Clean Architecture)
├── frontend/     # Angular 17+ (standalone, PrimeNG, TailwindCSS, Signals)
├── docs/         # Documentação de produto e desenvolvimento
└── CLAUDE.md     # Contexto do projeto pra Claude Code
```

> `backend/` e `frontend/` ainda não existem — serão criados nas Fases de setup (ver `docs/DESENVOLVIMENTO.md`).

## Documentação

Tudo em `docs/`:

- **`PROJETO.md`** — documento principal completo
- **`ENTREVISTA.md`** — descoberta de produto consolidada (fonte autoritativa de decisões de produto)
- **`DESENVOLVIMENTO.md`** — plano de desenvolvimento em fases com checklist
- **`FUTURO.md`** — visão de longo prazo / backlog pós-V1
- **`SEMANA.md`** — planejamento da semana corrente
- **`fluxos/`** — diagramas e fluxos de uso
- **`mocks/`** — protótipos e mocks de UI
- **`reuniões/`** — atas e notas de reunião

## Stack

- **Backend:** .NET 8 + ASP.NET Core Web API
- **Frontend:** Angular 17+ + PrimeNG + TailwindCSS
- **Banco:** Supabase (PostgreSQL)
- **IA:** Google Gemini API
- **Deploy:** Vercel (front) + Railway (back) — a confirmar na Fase 6
