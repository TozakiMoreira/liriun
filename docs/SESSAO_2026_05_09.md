## 1. i18n EN ~100% app interno

Pedido: "quero que o aplicativo todo esteja na opção de inglês pra poder ser usado". User reclamou várias vezes de strings PT residuais — fizemos varredura ampla.

### Telas + componentes traduzidos
- **Atividade (parabenização):** hero pts/nível/streak, esta semana, marcos, conquistas (6), frases positivas (8), toast, sub-tabs
- **Visão geral / Today:** saudação dinâmica, 5 mensagens contextuais, stats cards + subtítulos ("Tudo no prazo", "Bom ritmo", "no total"), agenda hoje, dias semana chart, finanças widget, listas hoje/feitas
- **Captura / Speak:** saudação, prompt, modos Manual+Liriun, sugestões, atalhos teclado, 12 aria-labels gravação, breadcrumbs, "Me conta", placeholders (mobile/desktop/ativo), header dinâmico
- **Tarefas:** view tabs, stats, filtros 2x (desktop+mobile dialog), kanban headers (Urgente/Importante/Normal/Baixa), semana view (dias + "Hoje" + rotuloDiaSelecionado), datas relativas (formatarPrazo + frasePeriodo), aria-labels (Próxima/Semana anterior), filtros_btn 2 lugares, mobile stat "Total"→"Pending"
- **Configurações:** section headers, Perfil completo, Aparência (cards + checkbox seguir sistema), Idioma, Zona perigosa, Excluir conta modal completo
- **tarefa-form modal:** labels, placeholders, prioridade chips, recorrência, observações, validações
- **tarefa-detalhe modal:** status badges, datas relativas, ações
- **Confirm modals** (excluir tarefa)
- **Sobre mockup:** Falar/Hoje/Tarefas/Histórico via `nav.*` keys

### Helpers datas i18n
- `data.hoje/ontem/amanha/em_n_dias`
- `dia.dom-sab` curto + `dia.domingo-sabado` completo
- `mes.jan-dez` (PT) / Jan-Dec (EN)
- `tarefas.em/atras/dias_lc/mes_1/meses_n/ano_1/anos_n` (frase período)

### Backend Gemini i18n
- `ContextoConversa.Idioma` (default `pt`)
- Front `IaService` envia `idioma: locale.locale()` em conversar + audio
- Controller recebe via body/form, propaga ao use case
- `MontarInstrucaoOneShot` injeta system prompt EN: "Respond only in English. Field 'mensagem' in English. 'titulo' keeps user's original language."
- User EN agora recebe respostas IA em inglês

### Erros HTTP
Todas chamadas `extrairProblemDetails` recebem `errors.sem_conexao` traduzido + fallback i18n
- ~10 lugares: tarefas (3x), concluidas (2x), captura (3x), tarefa-form, configurações (4x)
- Chaves: `errors.sem_conexao/fallback/salvar/carregar_tarefas/concluir_tarefa/excluir_tarefa/reabrir_tarefa/processar_ia/processar_audio/form_*`
- problem-details.ts assinatura: `extrairProblemDetails(err, fallback, semConexaoMsg?)`

### Effects reativos a locale
Limpa erros + recompute PageHeader on locale change
- Configurações, Tarefas, Concluidas, Captura, Visão Geral
- Effect dispara on `locale.locale()` change → reseta erros stale + reaplica pageHeader

### Aviso shell EN removido
Interface agora 100% EN, mensagem "interface in Portuguese only" ficou desatualizada.

### Total chaves novas em translations.ts
~280+ PT + EN (~560 entradas).
