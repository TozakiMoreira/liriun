## 3. Objeto do contrato

O Liriun é um **software como serviço (SaaS)** que permite ao usuário cadastrar tarefas, organizá-las por categoria, prioridade e prazo, e usar inteligência artificial (Google Gemini) para extrair tarefas a partir de texto livre ou áudio.

A versão V1 é oferecida **gratuitamente**. A gratuidade não afasta a aplicação do CDC quando o usuário se enquadrar como consumidor.
